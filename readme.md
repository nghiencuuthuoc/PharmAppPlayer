# PharmAppPlayer v16.5.x

PharmAppPlayer is a **portable offline video & lecture player** designed for:
- Traditional medicine (TCM/YHCT) training
- Internal courses and professional education
- Managing large structured video folders
- Remembering last watched position, subtitles, and quick search

No complex installation. Just one `.exe` file.

---

## 1. Installation

1. Create a folder for the player, for example:
   ```text
   D:\PharmAppPlayer\
````

2. Copy the executable file into this folder:

   ```text
   PharmAppPlayer_v16.5.x.exe
   ```

3. Prepare your video library folder, for example:

   ```text
   D:\YHCT_VIDEO\
   ```

   Inside, you can organize by subfolders:

   ```text
   D:\YHCT_VIDEO\duocthucdung1\
   D:\YHCT_VIDEO\duocthucdung2\
   ...
   ```

4. (Optional) Put subtitle files `.srt` or `.vtt` with the **same name** as the video file
   so the player can load them automatically.

---

## 2. Start & Select Folder

1. **Launch the player**
   Double–click `PharmAppPlayer_v16.5.x.exe`.

2. **Choose the video folder**

   * Click **“Open Folder [Ctrl+O]”** (top left).
   * Select the main folder with your videos (e.g. `D:\YHCT_VIDEO`).
   * PharmAppPlayer will scan and show all videos in the table below.

3. **Play a video**

   * Click any row in the table, then press **Play** or double–click the row.
   * The current title, time, and subtitle will appear above the list.

---

## 3. Main Controls

### Left Controls

* **Open Folder [Ctrl+O]** – Choose video root folder.
* **Open static.json [Ctrl+J]** – Open playlist/config file (for admins).
* **Rescan [F5]** – Rescan folder after adding/removing videos.
* **Set Image [Ctrl+I]** – Set a fixed background image.
* **Random Img** – Show a random background image.

### Right Controls

* **Prev [Ctrl+←]** – Previous video.
* **Play/Pause [Space]** – Start / Pause.
* **Next [Ctrl+→]** – Next video.
* **Stop [S]** – Stop playback.
* **Repeat [R]** – Toggle repeat.
* **Shuffle [H]** – Toggle shuffle playback.

### Center Bar

* **Search** – Filter videos by keyword in the title.
* **Sort** – Sort list (e.g. Title A–Z).
* **Folder (dropdown)** – Quickly switch between subfolders.
* **Vol + slider** – Adjust volume.
* **Static write** – For administrators to allow saving playlist/config.

### Subtitles

* **Load Sub [Ctrl+L]** – Manually load a subtitle file.
* **Sub: On [Ctrl+K]** – Turn subtitles On/Off.
* **A- [Ctrl+-] / A+ [Ctrl+=]** – Decrease/Increase subtitle font size.
* **Delay -5s / Delay +5s** – Shift subtitles earlier/later for sync.

### Playlist Table

Columns:

* **#** – Index
* **Title** – Video / lecture title
* **Folder** – Source folder
* **Duration / Size / Modified** – Useful info for library management

---

## 5. Recommended Use for Training Centers / Clients

* Use one dedicated folder per course (e.g. `YHCT_VIDEO`, `HERBAL_MED`, `INTERNAL_TRAINING`).
* Use clear file names, for example:

  * `B01_Intro_to_Traditional_Medicine.mp4`
  * `B02_Preparation_of_Medicinal_Herbs.mp4`
* Deliver to learners:

  * Course folder(s) + `PharmAppPlayer_v16.5.x.exe` + this `README_EN.md`.
  * Users just double–click the `.exe` and start learning. No installation.
